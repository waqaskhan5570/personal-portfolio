## Getting Started

First, run the development server:

# waqas-space-portfolio

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

This is a Next.js project for Waqas' portfolio website. It utilizes Next.js version 13 along with various dependencies for building a modern web application. Yarn is used as the package manager.

## Prerequisites

- Node.js >= 18.18.2
- npm >= 9.8.1

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/waqas-space-portfolio.git

2. Navigate into the project directory:

   ```bash
    cd waqas-space-portfolio

3. Install dependencies using NPM:
    ```bash
    npm install

## Scripts

- `dev`: Runs the development server.
- `build`: Builds the production-ready application.
- `start`: Starts the production server.
- `lint`: Lints the project files.



You can run these scripts using Yarn, for example:
`npm run dev`

## Usage

- Start the development server: `npm run dev`

- Build the production-ready application: `npm run build`

- Start the development server: `npm run start`

- Link the project files: `yarn lint`
